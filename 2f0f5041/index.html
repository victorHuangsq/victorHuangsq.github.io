华硕笔记本FL5900U装Win10专业版记录

发表于 2018-10-24 | 分类于 备忘记录
字数统计: 621 | 阅读时长 ≈ 2

以下内容纯属记录备忘，没有啥技术含量，可直接跳过。

背景

原先电脑买过来的时候默认装机Win10 家庭版，用了一段时间后，感觉不太习惯，想装双系统，一个跑Win7，一个跑Win10。尝试装Win7旗舰版的系统，发现装完之后，USB无法使用，后面网上找了一番知道这电脑没有Win7系统USB驱动。
没办法，将就用吧，不过用着用着感觉系统越来越慢，有时候CPU占用率非常的高。每次开机都要自动更新，关都关不掉。但是每次让它更新总是更新失败，总之各种问题。后面实在忍无可忍，决定安装Win10专业版系统。

尝试使用硬盘安装—失败

首先去系统之家下载装机版的iso和ghost软件。或者是系统城
但是，使用这种办法，启动按Esc选择启动项的时候会报告启动错误。如下：


总之，使用了好几个镜像都是卡在启动失败上，因此，这种方式不行。

U盘启动盘制作

  • 使用大白菜U盘启动盘制作工具，下载UEFI版本，按照UEFI教程进行启动盘制作。
  • 制作好之后，从itellyour下载Win10专业版iso镜像，将ISO镜像包复制到U盘的ISO文件夹，进入WINPE时会自动弹出一键安装程序。

但是在拷贝的时候提示文件太大。因为U盘使用的是FAT32系统，最大支持单个文件4G大小，因此要将其改为NTFS格式。解决方法参照：
使用u盘拷贝文件时提示对于目标文件系统文件过大怎么办.

BIOS设置

开机按F2进入BIOS，得到BIOS的界面如下：

需要在BIOS中做如下设置：

  • Secure Boot 设置为Disabled【禁用启动安全检查，这个最重要】
  • OS Optimized设置为Others或Disabled【系统类型设置】
  • CSM(Compatibility Support Module) Support设置为Yes或Enabled
  • UEFI/Legacy Boot选项选择成Both
  • UEFI/Legacy Boot Priority选择成UEFI First
