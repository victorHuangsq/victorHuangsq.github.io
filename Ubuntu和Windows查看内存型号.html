Ubuntu和Windows查看内存型号

发表于 2018-09-03 | 分类于 ubuntu工具
字数统计: 617 | 阅读时长 ≈ 3

有时候为了新增内存条，需要查看电脑使用的内存条型号。本文记录了如何在Ubuntu和Windows系统下不安装第三方工具查看内存条型号。

Ubuntu系统下查看内存条型号

使用sudo dmidecode -t memory命令，可以看到当前主板上支持4个卡槽，DIM1和DIM2分别插上了Micron的8GB DDR。

Windows系统下查看内存条型号

ctrl + R调出运行窗口，输入cmd进入命令行窗口，在命令行窗口输入wmic和memorychip就可以查看当前插上内存条的情况。
